# quiz-app

# Clone this project
$ git clone https://github.com/Dev-Pedrosv/quiz-app.git
# Access
$ cd quiz-app
# Install dependencies
$ yarn or npm 
# Run the project
$ yarn dev or npm start 
# The server will initialize in the <http://localhost:5173>
